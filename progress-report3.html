Progress Update 3 (10/25/2021)

Following feature extraction (preprocessing), the actual machine learning aspect of the project comes in. Since I'm using feature extraction, I know that I'm using some form of CNN. I plan on analyzing features from 24 hours before a flare event, meaning my features will also be sequential, or a time series. This leads me to believe that a CNN-LSTM would be most optimal for my purposes; however, I do plan on using many other models as well since some of the researches I read had decent results with different models. I'm specifically classifying whether or not a solar flare occurs, as well as the intensity of the flare since that directly affects us on Earth. So, this is a multiclass classification problem. I've looked at One-vs-Rest and One-vs-All methods of classification, and both methods have pros and cons, so I plan on trying both methods using this tutorial.
